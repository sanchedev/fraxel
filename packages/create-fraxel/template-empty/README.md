# Fraxel — Empty Template

A minimal Fraxel project ready to start building a 2D game.

## Getting Started

```bash
npm install # or pnpm install or yarn install
npm run dev # or pnpm dev or yarn dev
```

## Project Structure

```
src/
├── main.tsx          # Entry point — creates the game instance
├── scenes/
│   └── main.tsx      # Default scene
└── style.css         # Base styles
```
