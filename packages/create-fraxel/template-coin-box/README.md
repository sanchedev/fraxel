# Fraxel — Coin Box Template

A small drag-and-drop Fraxel game. Drag all five coins into the box to win.

## Getting Started

```bash
npm install # or pnpm install or yarn install
npm run dev # or pnpm dev or yarn dev
```

## Project Structure

```
src/
├── actions.ts          # Keyboard actions
├── main.tsx            # Entry point — creates the game instance
├── scenes/
│   ├── game.tsx        # Coin collection scene
│   ├── main.tsx        # Start screen
│   └── win.tsx         # Congratulations screen
└── style.css           # Base styles
```
